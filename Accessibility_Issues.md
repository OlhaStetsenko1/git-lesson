### Risk Description

**Title:** Accessibility Issues

**Detailed Risk Description:** Users may face difficulties accessing ELITEA without EPAM's VPN due to network or security configurations. This risk impacts the accessibility of the platform, potentially limiting user access and usability. The affected feature is the Nexus Elitea Environment, which is designed to be accessible without EPAM's VPN for all Epamers.

**Risk Category:** Infrastructure

**Risk Trigger:** Technology

**Risk Likelihood:** Middle

**Risk Impact:** High

**Responsible Person:** IT Infrastructure Manager

**Treatment/Maintenance Plan:**
1. Conduct a thorough review of network and security configurations.
2. Implement necessary changes to ensure accessibility without VPN.
3. Regularly monitor access logs to identify and resolve access issues promptly.
4. Provide alternative access methods if VPN-free access is not feasible.

**Risk Strategy:** Mitigate

**Mitigation Plan Actions:**
1. Review and update network configurations.
2. Enhance security protocols to allow VPN-free access.
3. Test access from various locations and devices to ensure reliability.
4. Train support staff to handle access issues efficiently.

**Contingency Plan Actions:**
1. Provide temporary VPN access for users facing issues.
2. Set up a dedicated support line for immediate assistance.
3. Communicate alternative access methods to all users.
4. Regularly update users on the status of the issue and resolution efforts.
