### Test Team

The test team for the e-commerce platform MVP consists of the following members, each with specific roles and responsibilities to ensure the successful execution of the testing process.

#### Roles and Responsibilities

1. **Test Manager: John Doe**
   - Overall responsibility for the test strategy, planning, and execution.
   - Ensures that the testing process aligns with the project goals and objectives.
   - Manages resources, schedules, and budgets for the testing activities.
   - Communicates test progress, issues, and risks to stakeholders.

2. **Test Lead: Jane Smith**
   - Leads the test team and coordinates testing activities.
   - Develops detailed test plans, test cases, and test scripts.
   - Ensures that test cases cover all functional and non-functional requirements.
   - Reviews test results and ensures that defects are properly logged and tracked.

3. **Test Engineers:**

   - Execute test cases and report defects.

4. **Automation Test Engineers:**
