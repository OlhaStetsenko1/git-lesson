# List of the Metrics to Track on a Project

| Metric name | Formula | Responsible role | Frequency | Way to provide |
|-------------|---------|------------------|-----------|----------------|
| Defect Containment (DC) | DC = (Number of non-rejected defects registered during by end-users on production environment) / (Total number of non-rejected defects registered) | Testing Team Lead | After each release to production | 2nd column chart |
| Test Case Execution Rate | (Number of executed test cases / Total number of test cases) * 100 | Test Manager | Daily during test execution phase | Test management tool dashboard |
| Defect Density | (Total number of defects / Size of the software module) | Quality Assurance Lead | After each release | Defect tracking tool report |
| Test Coverage | (Number of covered requirements / Total number of requirements) * 100 | Test Analyst | Weekly | Test management tool report |
| Mean Time to Detect (MTTD) | Sum of time taken to detect each defect / Total number of defects | Test Lead | After each release | Test management tool report |
| Mean Time to Resolve (MTTR) | Sum of time taken to resolve each defect / Total number of defects | Development Lead | After each release | Defect tracking tool report |
| Requirement Stability Index | (Number of initial requirements - Number of changes in requirements) / Number of initial requirements | Business Analyst | After each release | Requirement management tool report |
| Test Case Pass Rate | (Number of passed test cases / Total number of executed test cases) * 100 | Test Manager | Daily during test execution phase | Test management tool dashboard |
| Defect Rejection Rate | (Number of rejected defects / Total number of reported defects) * 100 | Quality Assurance Lead | After each release | Defect tracking tool report |
| Automation Coverage | (Number of automated test cases / Total number of test cases) * 100 | Automation Lead | Weekly | Test automation tool report |
